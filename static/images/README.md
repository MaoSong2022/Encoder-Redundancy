# Project page images

Figures used by the project page are stored here:

- **Redundancy_illustration.jpg** – Main teaser: illustration of encoder redundancy (converted from PDF).
- **all_models_scores.png** – Performance of multi-encoder MLLMs with different numbers of masked encoders.
- **cur_results.png** – Conditional Utilization Rate (CUR) across benchmark categories.
- **long_case_1.jpg**, **long_case_2.jpg** – Case studies (Eagle-X4 8B with/without encoders masked; converted from PDF).

To update figures, replace these files from your paper build (e.g. from `iclr2026/figures/`). For social preview (Open Graph/Twitter), the page uses `all_models_scores.png`; you can add a dedicated 1200×630px `social_preview.png` if desired.
